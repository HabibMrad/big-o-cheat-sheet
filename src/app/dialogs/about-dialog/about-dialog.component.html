Note about elementary operations

Unless stated otherwise, elementary operations such as adding two numbers, multiplying two numbers, and accessing a location in memory are assumed to run in constant time. Additionally, it is assumed that any single number or address occupies constant space in memory. This assumption greatly simplifies the time complexity analysis.
